# AI Agent Platform Frontend

Production-ready React console for interacting with FastAPI agents.

## Stack

- React + Vite + TypeScript
- TailwindCSS
- Framer Motion
- Axios

## Setup

1. Install dependencies:

```bash
npm install
```

2. Configure environment:

```bash
cp .env.example .env
```

Set `VITE_AGENT_ID` to a valid backend agent ID.

3. Run development server:

```bash
npm run dev
```

Frontend default URL: `http://localhost:5173`  
Backend expected URL: `http://localhost:8000`
